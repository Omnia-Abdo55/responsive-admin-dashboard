# 📊 Responsive Admin Dashboard

A clean and modern **Admin Dashboard UI** built using **HTML5** and **CSS3**.

This project demonstrates a responsive dashboard layout with a collapsible sidebar, statistic cards, and a product table. It focuses on creating an organized and visually appealing user interface suitable for admin panels.

## ✨ Features

* 📱 Responsive Layout
* 📂 Collapsible Sidebar Navigation
* 📊 Dashboard Statistics Cards
* 📋 Product Management Table
* 🎨 Modern UI Design
* 💻 Clean HTML & CSS Structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome
